# Resume Website

This repository contains a one-page dark themed resume built using HTML and CSS. It is designed for internship applications, specifically tailored for a Software Engineering internship.

## Features

- Dark theme design
- Clean, professional layout
- Responsive for mobile and desktop
- Easy to export as PDF

## Live Demo

My site is live at https://nancy-githinji.github.io/Resume/


## Files

- `resume.html` – The main resume page
- `style.css` – Styling for the resume

## How to run locally

1. Clone the repository
2. Open `resume.html` in your browser
